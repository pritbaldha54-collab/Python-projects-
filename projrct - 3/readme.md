# 🎓 Student Record Management System

A simple command-line Python application for managing student records. This project allows users to add, view, update, and delete student information, as well as display all available subjects.

---

## 📌 Features

- ➕ Add Student
- 📋 Display All Students
- ✏️ Update Student Information
  - Update Name
  - Update Age
  - Add New Subject
- 🗑️ Delete Student
- 📚 Display All Subjects
- 🚪 Exit Program

---

## 🛠️ Technologies Used

- Python 3
- List
- Dictionary
- Set
- Loops
- Match-Case
- User Input

## ▶️ How to Run

1. Install Python 3.x.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

```bash
python Index.py
```
